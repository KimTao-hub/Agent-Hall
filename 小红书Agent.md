# 小红书Agent 功能与实现原理

## 1. 核心功能

### 1.1 场景覆盖
- **美妆护肤**：产品测评、使用体验、效果分析
- **时尚穿搭**：风格搭配、服装推荐、穿搭技巧
- **旅行攻略**：目的地指南、行程规划、实用贴士
- **美食探店**：餐厅推荐、菜品测评、用餐体验
- **家居好物**：产品推荐、使用体验、空间优化
- **健身运动**：运动方案、效果分享、注意事项
- **育儿分享**：育儿经验、产品推荐、问题解决
- **科技数码**：产品测评、使用体验、性能分析

### 1.2 功能模块
1. **场景选择**：直观的场景导航界面，支持8大核心场景
2. **参数配置**：场景化的参数表单，根据不同场景提供定制化输入字段
3. **智能生成**：基于AI的文案生成引擎，根据场景和参数生成高质量小红书风格文案
4. **文案编辑**：实时编辑功能，支持标签管理和格式调整
5. **效果预览**：小红书风格的实时预览，支持一键复制

## 2. 实现原理

### 2.1 技术架构
- **前端**：React + Next.js + TypeScript
- **后端**：Node.js + Express + TypeScript
- **AI集成**：OpenAI SDK (集成DeepSeek API)
- **数据流**：前端 → 后端API → AI服务 → 流式响应 → 前端展示

### 2.2 核心流程
1. **场景选择**：用户从8大场景中选择目标场景
2. **参数配置**：根据选择的场景，展示对应的参数表单，用户填写相关信息
3. **请求处理**：前端将场景和参数发送到后端API
4. **AI调用**：后端根据场景生成对应prompt，调用OpenAI API生成文案
5. **流式响应**：使用Server-Sent Events (SSE) 技术，实时返回生成的文案
6. **前端展示**：接收流式数据，实时更新编辑区域和预览区域
7. **编辑优化**：用户可对生成的文案进行编辑和优化
8. **效果预览**：实时展示小红书风格的文案效果，支持一键复制

### 2.3 技术亮点
- **场景化prompt**：为每个场景设计专属的prompt模板，确保生成的文案符合场景特点
- **流式响应**：减少用户等待时间，提升交互体验
- **响应式设计**：适配不同屏幕尺寸，提供良好的移动端体验
- **模块化架构**：前后端分离，组件化设计，易于维护和扩展
- **实时预览**：提供小红书风格的实时预览，帮助用户快速调整文案

## 3. 使用方式

1. **启动服务**：
   - 后端：`cd backend && pnpm install && pnpm run dev`
   - 前端：`cd frontend && pnpm install && pnpm run dev`

2. **访问界面**：打开前端服务地址（默认 http://localhost:3000）

3. **生成文案**：
   - 选择目标场景
   - 填写场景参数
   - 点击生成按钮
   - 编辑和优化生成的文案
   - 预览效果并复制使用

## 4. 应用价值

- **提升效率**：快速生成高质量小红书文案，节省创作时间
- **保证质量**：基于AI技术，生成符合平台风格的优质内容
- **场景覆盖**：支持8大核心场景，满足多样化创作需求
- **用户友好**：直观的操作界面，降低创作门槛
- **定制化服务**：根据不同场景提供定制化的文案生成服务

## 5. 技术创新

- **场景化AI应用**：将AI技术与特定场景深度结合，提供更精准的服务
- **实时交互体验**：通过流式响应技术，实现实时生成和展示
- **多维度参数配置**：为每个场景设计专属参数，提升文案针对性
- **全流程解决方案**：从场景选择到文案预览，提供完整的创作流程
- **平台风格适配**：生成的文案符合小红书平台的语言风格和内容特点
